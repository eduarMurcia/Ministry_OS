# Proyecto: Ministry OS

## Qué es
Plataforma modular de administración y producción de contenido para ministerios cristianos. Ambición de largo plazo: plataforma completa (cubre tanto lo operativo como la capa creativa/de conocimiento). Se construye primero para un caso real: la Primera Iglesia Bíblica Bautista de Montenegro (Quindío), con un equipo pequeño.

## Estado (2026-07-03)
Fase de definición de alcance, sin desarrollo de software todavía. Repositorio en `github.com/eduarMurcia/Ministry_OS`, con primer commit (Mapa de Capacidades + Alcance v1).

## Alcance v1 cerrado
- Administración: directorio de personas (miembros/nuevos/visitantes), informes contables (sin gestión de dinero), orden de servicio (basado en plantilla real), asignación de responsables, check-in de niños, grupos pequeños, eventos gratuitos con documentación propia, calendario+tareas (con integración deseada a Google Calendar).
- Comunicación: exclusivamente WhatsApp (ya existe comunidad con grupos).
- Publicaciones: rol de "equipo de marketing" para canal, RRSS y página web existentes.
- Fuera de v1: formularios públicos, portal de autoservicio, pagos online.
- Visión a futuro (capa creativa): preparación de sermones (manuscrito + PowerPoint, con plantilla real de revisión para publicación), diseño gráfico segmentado por ministerio (niños, ancianos, damas, caballeros, jóvenes, evangelismo, tratados).

## Insumos reales disponibles
- Plantilla de Orden de Servicio Dominical (bulletin semanal).
- Plantilla de Sermón escrito (manuscrito con texto clave, introducción, exposición, cierre institucional).
